Attack Targets

The targets typically include various network and system resources that can be overwhelmed to disrupt normal operations. Here are some common targets:

Web Servers

HTTP Flood: Overloading a web server with HTTP requests to exhaust its resources, making it unavailable to legitimate users.

Application Servers

Application-level Attacks: Targeting specific applications running on a server to exploit vulnerabilities and cause excessive resource consumption.

Network Infrastructure

Bandwidth Consumption: Saturating the available network bandwidth with excessive traffic, preventing legitimate traffic from reaching its destination.
Router and Switch Overload: Sending a high volume of packets to routers and switches to exhaust their processing capabilities.

DNS Servers

DNS Flood: Sending a large number of requests to a DNS server to overwhelm it and prevent it from resolving domain names.
